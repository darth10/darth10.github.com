programming [mass noun] /'prəʊɡramɪŋ/
  1. the planning, scheduling, or performing of a program
  2. the process of instructing or learning by means of an instructional program
  3. the process of preparing an instructional program

hack [intransitive verb] /'hak/
  1. to write computer programs for enjoyment
  2. to gain access to a computer illegally
