The First Thing to Do...

May 13th, 2009

I thought of writing a py script to help with sharing blog stuff, so here goes. The usage of os.mkdir() is obvious, but to name the directory uniquely a lot of strategies are possible. I just figured as I would blog at a daily basis at best, just use time.gmtime().

#!/usr/bin/python
import os
from time import strftime,gmtime

date = strftime("%a.%d.%b%Y", gtime())
try:
    os.mkdir(date, 0700)
except OSError:
    print "Haven't you already done this today?"
else:
    print "Created directory ", date

So that basically does it I guess; nothing too fancy. If you have any better ideas about solving the unique directory problem please leave your comments.
